# System Health Monitor (Bash)

This project is a Bash script that monitors system health and helps in basic Linux debugging.

## 🔧 Features

- Checks if a service is running (auto-restarts if down)
- Monitors disk usage
- Displays memory usage
- Shows top CPU-consuming processes
- Logs all results into a report file

## ⚙️ Tech Used

- Bash scripting
- Linux commands (ps, df, free, systemctl, ss)

## 🚀 Use Case

Built to simulate real-world infrastructure monitoring and debugging scenarios.
